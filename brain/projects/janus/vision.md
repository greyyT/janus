# Janus Vision

Janus is a personal engineering memory that stays Markdown-first and easy to maintain.

## Principles

- Capture should be cheap.
- Durable knowledge should be deliberate.
- Source repositories remain authoritative for code behavior.
- Generated indexes are disposable and lower authority than source Markdown.
- Tooling should support maintenance, not become a product surface.

## v0 direction

The root is optimized for capture. `brain/` is optimized for retrieval and durability.

A root inbox note can remain rough, incomplete, or unverified. Promotion into `brain/` is the point where Janus decides the note is worth preserving as durable engineering memory.
